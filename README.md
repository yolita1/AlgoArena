# AlgoArena 


AlgoArena permet donc de s'entrainer académiquement de maniére plus stimulante, le principe étant de réviser en jouant contre d'autres éléves d'autres établissements, une compétitivité stimulante qui pousse à l'amélioration.

## Tuto

```bash
# prérequis: Node.js ≥ 18, gcc, python3, ocaml

chmod +x start.sh && ./start.sh    # Linux / macOS
start.bat                           # Windows
```

puis ouvre**http://localhost:5173**
